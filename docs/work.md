# 推荐协作边界

两个人之间需要统一以下接口，避免后期合并困难。

## 1. 数据库表结构由成员 A 确定

成员 B 按照成员 A 提供的 SQL 表结构写 C++ 代码。

成员 A 需要交付给成员 B：

```text
建表 SQL
测试数据
字段说明文档
核心查询 SQL
数据库连接信息
```

---

## 2. C++ 调用 SQL 的接口由成员 B 封装

成员 B 将数据库操作封装成函数，例如：

```cpp
std::vector<Member> findChildren(int memberId);
std::vector<Member> findAncestors(int memberId);
std::vector<Member> fuzzySearchMember(int genealogyId, QString keyword);
std::vector<int> findRelationPath(int genealogyId, int memberA, int memberB);
```

成员 A 只需要保证对应 SQL 可以正确执行。
